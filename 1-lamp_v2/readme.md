## 💡 Lamp

/<img src="source/view1">

This project is a reconstruction of one of my first experiences with JavaScript.

It serves as a great starting point to revisit fundamental concepts and regain familiarity with the language. At this stage, the focus is on rebuilding older projects with improved quality and intention.

In this version, all visual elements were created using CSS, unlike the original approach, which relied on switching between images to represent the lamp being on and off.

Reference used:
https://youtu.be/Gy2BP857030

Some adaptations were made, including responsive versions for mobile testing. The goal is for this project to be part of a more complete portfolio page, where projects are presented similarly to apps in an app store, with a stronger emphasis on visual appeal and organization.

Este projeto é a reconstrução de um dos meus primeiros contatos com JavaScript.

A ideia é que este projeto se integre a uma página de portfólio mais completa, onde os projetos serão apresentados de forma semelhante a aplicativos em uma loja, priorizando uma apresentação visual mais atraente e organizada.
